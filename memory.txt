[MAPA PAMIĘCI]

$00-$7f         zero page for OS (base page, coz you need only 1 byte for accessing to it)
$80-$ff         zero page for you

$100-$1ff       CPU stack

$200-$2ff       OS vectors and registers
$300-$3ff       OS vectors, registers, and control/IO blocks

$400-$5ff       OS buffers (e.g. casette buffer, keyboard buffer, floating-point operations buffer and so on)

$600-$6ff       Page 6, object code in BASIC

$700-$1fff      Disc Operating System (using disc drive)

$2000-RAMTOP    Your code ($1000 = 4kB)

$8000-$9fff     Cartrige 8k/low 16k
$a000-$bfff     Cartrige 8k/high 16k/BASIC

$c000-$cfff     OS part 1
$c000-$c3ff     OS character set, international (XE/XL)

$d000-$d0ff     GTIA (pixels, colors, sprites, missiles, consol, beep)
$d100-$d1ff     PBI
$d200-$d2ff     POKEY (paddle, keyboard, SIO, audio, random number generator)
$d300-$d3ff     PIA (joystick, MMU)
$d400-$d4ff     ANTIC (DMA feed GTIA with data)
$d500-$d5ff     Module port (R/W)
$d600-$d7ff     Reserved (VBXE, SIDE, ... more extensions)

$d800-$dfff     Floating Point Package (FPP)

$e000-$ffff     OS part 2
$e000-$e3ff     OS character set, standard (1kB, $3ff=1023 + $000)
